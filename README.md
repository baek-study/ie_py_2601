## 첫 파이썬 코드 (Hello World)
* `print()` 함수 : 괄호 안의 데이터나 문자를 화면에 출력하는 기능
* **문자열 출력:** 텍스트를 출력할 때는 반드시 큰따옴표(`""`)나 작은따옴표(`''`)로 감싸주어야 함
  * `print("Hello World")`
  * `print("백혜정")`
